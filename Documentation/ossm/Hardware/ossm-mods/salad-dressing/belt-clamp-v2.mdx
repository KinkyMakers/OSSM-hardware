This is a proposed new standard part that addresses printability and strength issues in the current standard belt clamp design.
</Note>

## Features

- **MGN12 rail compatibility**: Slotted design fits MGN12 linear rails
- **Wide belt support**: Pass-through accommodates belt widths up to 15mm
- **Universal thread tolerance**: Fine-tuned threading works with unmodified 24mm female threaded parts regardless of printer tolerance

## Print settings

<Note>
Print all parts without supports using OSSM standard settings.
</Note>

| Setting | Value |
|---------|-------|
| Walls | 6 |
| Infill | 15% crosshatch or gyroid |
| Supports | None |
| Estimated print time | ~3 hours |
| Estimated filament | ~85g |
