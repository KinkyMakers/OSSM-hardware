The OSSM remote control provides a convenient handheld interface for controlling your device. This guide covers the print settings and hardware you need to build the remote enclosure.

## Printed Parts

You need to print the following components:

| Part | Quantity |
|------|----------|
| Body | 1 |
| Top Cover | 1 |
| Knob | 2 |

<Tip>
Additional knob designs are available in the [Non-standard](Non-standard/) directory if you prefer alternative styles.
</Tip>

## Print Settings

Use these settings for all remote parts:

| Setting | Value |
|---------|-------|
| Walls | 3 |
| Infill Type | Cross Hatch or Gyroid |
| Infill Percentage | 25% |
| Supports | Yes - Auto |

<Note>
Estimated filament usage is approximately **70g** for all parts.
</Note>
