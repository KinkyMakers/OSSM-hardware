The OSSM wired remote control provides a convenient handheld interface for controlling your device. This guide covers the print settings and hardware you need to build the wired remote enclosure.

<Info>
**Looking for wireless control?** An alternative wireless remote called **RADR** is available as an open source controller for the OSSM. [Learn more about RADR](/radr/) for a cable-free experience.
</Info>

## Printed Parts

You need to print the following components:

| Part | Quantity |
|------|----------|
| Body | 1 |
| Top Cover | 1 |
| Knob | 2 |

<Tip>
Additional knob designs are available in the Non-standard directory (in the source repository) if you prefer alternative styles.
</Tip>

## Print Settings

Use these settings for all remote parts:

| Setting | Value |
|---------|-------|
| Walls | 3 |
| Infill Type | Cross Hatch or Gyroid |
| Infill Percentage | 25% |
| Supports | Yes - Auto |

<Note>
Estimated filament usage is approximately **70g** for all parts.
</Note>
