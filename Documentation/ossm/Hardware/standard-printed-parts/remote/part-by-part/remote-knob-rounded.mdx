## Function

The Rounded Knob is the default press-fit knob for the wired OSSM remote's rotary encoders. This component:

- Smooth, rounded profile for comfortable fingertip control
- Press-fits onto the rotary encoder shafts
- Provides tactile feel for speed and stroke adjustments

<Tip>
Material: PLA or PETG (PETG recommended for durability). Alternate knob shapes (knurled, simple) are available in the "Non-standard" folder.
</Tip>
