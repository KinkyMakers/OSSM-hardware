## Function

The Rounded Knob is the default control knob for the remote. This component:

- Press-fits onto the rotary encoder shafts
- Features a smooth, rounded profile for comfortable finger control
- Provides tactile feedback when adjusting speed and stroke settings
- Two knobs are required per remote (one for each encoder)

<Tip>
Additional knob designs are available in the Non-standard directory if you prefer alternative grip styles like knurled or simple profiles.
</Tip>
