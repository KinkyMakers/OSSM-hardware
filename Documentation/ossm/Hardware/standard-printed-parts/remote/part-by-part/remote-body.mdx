## Function

The Remote Body is the main housing for the wired remote control. This component:

- Encloses the remote control PCB and electronics
- Features mounting bosses for the rotary encoders (knobs)
- Provides the hand-held ergonomic grip surface
- Accepts the Top Cover to complete the enclosure

## Print Settings

| Setting | Value |
|---------|-------|
| Walls | 3 |
| Infill Type | Cross Hatch or Gyroid |
| Infill Percentage | 25% |
| Supports | Yes - Auto |

<Note>
Estimated filament usage for all remote parts is approximately **70g**.
