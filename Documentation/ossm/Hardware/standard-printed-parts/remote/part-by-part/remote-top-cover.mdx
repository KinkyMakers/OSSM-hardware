## Function

The Remote Top Cover completes the remote control enclosure. This component:

- Provides protective covering for the internal electronics
- Features openings for the rotary encoder knobs
- Secures to the body with M4x6 button cap bolts
- Creates a finished appearance for the handheld remote

## Print Settings

| Setting | Value |
|---------|-------|
| Walls | 3 |
| Infill Type | Cross Hatch or Gyroid |
| Infill Percentage | 25% |
| Supports | Yes - Auto |

## Hardware Required

| Quantity | Part |
|----------|------|
| 2 | M4x6 Button Cap Bolt |
