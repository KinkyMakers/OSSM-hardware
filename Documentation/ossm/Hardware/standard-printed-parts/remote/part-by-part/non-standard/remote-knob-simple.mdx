## Function

The Simple Knob is a minimalist alternative for the remote control. This component:

- Press-fits onto the rotary encoder shafts
- Features a clean, simple profile without texture
- Provides a smooth control surface for adjustment
- Two knobs are required per remote (one for each encoder)

<Note>
This is a non-standard part. The default build uses the Rounded Knob instead.
</Note>
