## Function

The Pivot Plate Left is one of two matching plates that enable angle adjustment. This component:

- Mounts to the 3030 aluminum extrusion frame
- Creates the left-side pivot point for the actuator arm
- Works with the Pivot Plate Right to provide synchronized angle adjustment
- Uses M6 hardware with rubber washers for friction-locked positioning

You need **1 Pivot Plate Left** and **1 Pivot Plate Right** per stand assembly.

## Print Settings

| Setting | Value |
|---------|-------|
| Walls | 6 |
| Infill Type | Cross Hatch or Gyroid |
| Infill Percentage | 25% |
| Supports | None |
