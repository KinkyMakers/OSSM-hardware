The 3030 Extrusion Stand is the default mounting solution for your OSSM. This design uses aluminum extrusion for a sturdy, adjustable base with a single pivot arm that allows angle adjustment during use.

## 3D printing the parts

Print all stand components using the settings below for optimal strength and durability. You can find the relevant print files under the Stand's "Part by Part" tab.

### Print settings

| Setting              | Value                 |
| -------------------- | --------------------- |
| Walls                | 6                     |
| Infill type          | Cross Hatch or Gyroid |
| Infill percentage    | 25%                   |
| Supports             | None                  |
| Filament estimate    | 246g                  |
| Recommended filament | PLA+                  |

<Tip>
  All parts are oriented correctly in the provided STL files. Print them in their default orientation without rotation.
</Tip>
