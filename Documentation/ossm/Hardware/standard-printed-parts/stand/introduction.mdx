The 3030 Extrusion Stand is the default mounting solution for your OSSM. Aluminum extrusion provides a sturdy, adjustable base with a single pivot arm for angle adjustment during use.

## Print Settings

| Setting              | Value                 |
| -------------------- | --------------------- |
| Walls                | 6                     |
| Infill Type          | Cross Hatch or Gyroid |
| Infill Percentage    | 25%                   |
| Supports             | None                  |
| Filament Estimate    | 246g                  |
| Recommended Filament | PLA+                  |

<Tip>
  All parts are oriented correctly in the provided STL files. Print them in their default orientation without rotation.
</Tip>

<Tip>
  All OSSM print files are also available on <a href="https://makerworld.com/en/models/2472952">MakerWorld</a>.
</Tip>
