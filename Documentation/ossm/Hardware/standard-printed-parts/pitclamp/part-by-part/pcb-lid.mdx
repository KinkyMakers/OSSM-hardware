## Function

The PCB Lid completes the electronics enclosure. This component:

- Provides protective covering for the OSSM control electronics
- Snaps or fastens to the PCB Box base
- Shields electronics from dust and accidental contact
- Creates a finished appearance for the control unit

## Print Settings

| Setting | Value |
|---------|-------|
| Walls | 6 |
| Infill Type | Cross Hatch or Gyroid |
| Infill Percentage | 10% |
| Supports | None |
