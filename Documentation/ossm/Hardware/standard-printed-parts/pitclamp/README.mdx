## Overview

**Developed by: armpit**

PitClamp Mini is a 3D-printed modular quick-release mount ecosystem designed for NEMA23 motors used in OSSM builds. The system has been stress tested to 50 ft-lb without grip compromise (PLA, 6 walls, 10% infill).

## Print Settings

Configure your slicer with these settings for optimal strength and fit.

| Setting | Value |
|---------|-------|
| Walls | 6 |
| Infill type | Cross Hatch or Gyroid |
| Infill percentage | 10% |
| Supports | Tree Supports, 5Â° threshold angle |
| Seam | Nearest (or your preference) |

<Warning>
Print all parts in the default file orientation. Printing base components in other orientations causes structural weakness.
</Warning>

<Tip>
If your printer is well-calibrated with optimal layer adhesion, use the print-in-place Upper+Handle variant from the Non-Standard folder.
</Tip>

## Features

### Quick-Adjust Rotation

The mount provides 4-degree precision quick-adjustable rotation, allowing you to fine-tune the motor angle without disassembly.

### Modular Base System

Your motor can be quickly removed from the mount for storage or transferred to another base. Choose from multiple base options:

<AccordionGroup>
<Accordion title="Standard Base">
- Mounts to 3030 or 4040 extrusion (top mount)
- 64mm M6 hole spacing
- ~30mm overall width
</Accordion>

<Accordion title="Quick Clamp Ratchet + 3030 Extrusion Mount">
Compatible with POWERTEC 71408 or similar guide rail MFT clamps.

<Tip>
Search for "Guide rail MFT ratchet clamp" to find compatible products.
</Tip>
</Accordion>

<Accordion title="Folding Mount">
Folds flat for easier frame storage, such as sliding under a bed.
</Accordion>

<Accordion title="Tabletop Clamp (Work-In-Progress)">
A fully printed tabletop clamp solution is under development.
</Accordion>
</AccordionGroup>

### Motor Cover and Electronics Support

<Frame caption="PitClamp Mini in action with motor cover">
<img src="./_images/PitClamp Mini - Action.webp" alt="PitClamp Mini mounted with motor cover installed" style={{width: "50%"}} />

The v1.1 Ring (available for 57AIM only) supports both M5 Hex Nuts and M5 Hex Coupling Nuts (20mm). When using Coupling Nuts, the ring features ridges for an interlocking connection to an optional rear motor cover.

<Check>
**Recommended setup:** Use M5 Hex Coupling Nuts (20mm) for motor cover compatibility.
</Check>

### Ring Offset Specifications

- Ring offset: +5mm from the face of the PitClamp Mini Base
- Standardized after clearance testing against various rail tensioners
- Compatible with both 3030 and 4040 extrusion alignment
