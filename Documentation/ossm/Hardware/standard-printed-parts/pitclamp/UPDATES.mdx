<Update label="v1.1" description="April 2, 2025">
## Improvements

- Pivot bolt changed from M6x20 to M5x20
- Lower Dogbone grip slightly loosened for easier assembly
</Update>

<Update label="Official Release" description="March 6, 2025">
## OSSM Project Integration

PitClamp Mini officially released as part of the OSSM project standard parts.
</Update>

<Update label="v1.1 Ring" description="November 22, 2024">
## New Ring Design

Added v1.1 57AIM Ring with improved mounting features. This is now the recommended default ring for 57AIM motors.
</Update>

<Update label="File Restructure" description="September 8, 2024">
## Organization Updates

- Refactored file structure for better organization
- Updated default files to simplify BOM
</Update>

<Update label="Helper Brim" description="September 4, 2024">
## Print Improvements

Added **Standard Base w/ Helper Brim** STEP file to Files folder. This helps prevent the latch corner from lifting during printing.
</Update>

<Update label="Ring Templates" description="September 3, 2024">
## Design Resources

- Added **Ring Template** STEP files to Ring Template folder
- Added **Centered 42AIM Ring** to Non-Standard Parts folder (compatible with just_b's motor head)
</Update>

<Update label="Cable Passthru" description="August 29, 2024">
## Experimental Features

Added **57AIM Cable Passthru** Ring to Non-Standard Parts folder for experimenting with motor head mounted Reference Board.
</Update>

<Update label="Print-In-Place" description="August 28, 2024">
## Reduced Hardware Option

- Added **Print-In-Place PitClamp Mini** to Experimental folder (reduces required hardware)
- Revised **Extrusion Ring** from Normal to Experimental status (needs stronger design for re-release)
</Update>
