## PitClamp Mini v1.1

<Frame caption="PitClamp Mini exploded view showing all components">
<img src="_images/Exploded%20-%20PitClamp%20Mini%20Base.png" alt="Exploded view of PitClamp Mini Base assembly" />
</Frame>

**Developed by: armpit**

A 3D-printed modular quick-release mount ecosystem for NEMA23 motors, featuring:

- 4-degree precision quick-adjustable rotation
- Motor can be quickly removed for storage or use in another base
- Stress tested to 50 ft-lb without grip compromise (PLA, 6 walls, 10% infill)

<Note>
The iHSV57 ring prints as one piece but snaps into two halves after removal from the printer. Clasp each half around the motor to assemble.
</Note>

<Warning>
iHSV57 motors smaller than 180w require modifications to the ring design.
</Warning>

## PCB Enclosure

Protective enclosure for the OSSM control board with direct mounting to 3030 extrusion.

### Print Settings

| Setting | Value |
|---------|-------|
| Walls | 4 |
| Infill type | Cross Hatch or Gyroid |
| Infill percentage | 10% |
| Supports | None |
| Recommended filament | PLA+ |

### Required Printed Parts

```
(Qty 1) PCB - Box
(Qty 1) PCB - Lid
```
