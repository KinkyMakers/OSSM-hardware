## Function

The Body - Cover completes the actuator enclosure. This component:

- Protects the timing belt from dust, debris, and contact
- Provides mounting points that secure to the Body - Middle
- Encloses the belt routing channel for safety
- Completes the aesthetic appearance of the actuator

<Warning>
Print this part in the default file orientation. Printing in other orientations causes structural weakness.
</Warning>
