## Function

Drop-in alternative to the standard 5-sided nut. This component:

- Internal 24 mm thread matches the OSSM clamping thread components
- Hex-flower profile provides more grip positions for comfortable hand tightening
- Works with the optional Hex Flower wrench for additional leverage
- Secures toy mounting accessories against the end effector

<Note>
This is a non-standard part. The default build uses the "24 mm Nut - 5-Sided" design.
</Note>

## Compatibility

- Interface: OSSM 24 mm clamping thread family
- Use cases: Locking vac-style adapters, harness inserts, and other end-effector accessories
- Tools: Optional Hex Flower wrench (manual use only; do not use power tools)

## Print Guidelines

- Material: PETG or ABS/ASA recommended for heat and impact resistance; PLA+ acceptable for prototypes
- Layer height: 0.2-0.28 mm
- Perimeters: 4-6 (thicker walls improve thread durability)
- Infill: 35-50% (gyroid or grid)
- Supports: Not required in the default orientation

<Warning>
Print in the default file orientation. Reorienting printed threads weakens layer adhesion.
</Warning>

<Tip>
Lightly deburr the lead-in of the internal thread with a hobby knife, then chase the threads by running the nut on and off a spare 24 mm mating thread a few times.
</Tip>
