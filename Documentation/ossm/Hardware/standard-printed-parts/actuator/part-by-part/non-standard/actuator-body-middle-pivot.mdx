## Function

The Body - Middle Pivot is an alternative to the standard middle body section. This component:

- Provides the same linear rail mounting and belt routing as the standard middle
- Adds a pivot point allowing angular adjustment of the front section
- Enables greater positioning flexibility in certain mounting configurations
- Useful for setups requiring non-standard actuator angles

<Note>
This is a non-standard part. The default build uses the standard Body - Middle instead.
</Note>

<Warning>
Print this part in the default file orientation. Printing in other orientations causes structural weakness.
</Warning>
