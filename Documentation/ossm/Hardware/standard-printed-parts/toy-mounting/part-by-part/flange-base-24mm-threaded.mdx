## Function

The Flange Base with 24mm Threading is a universal adapter for toy mounting. This component:

- Screws onto the 24mm threaded end effector of the actuator
- Provides a flat flange surface for mounting various toy adapters
- Compatible with dildo rings and other flange-mount accessories
- Secures with the 24mm Nut - 5 Sided against the end effector

This is a versatile base that accepts additional adapters like the dildo ring inserts.

<Warning>
Print this part in the default file orientation. Printing threaded components in other orientations causes structural weakness.
</Warning>
