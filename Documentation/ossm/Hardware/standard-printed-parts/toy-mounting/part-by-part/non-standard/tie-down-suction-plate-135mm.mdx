## Function

The 135mm Tie Down and Suction Plate provides a larger versatile toy mounting surface. This component:

- Offers a 135mm diameter mounting surface for larger toys
- Features slots for tie-down straps to secure toys
- Provides a smooth surface for suction cup attachment
- Supports dual mounting methods for different toy types

<Note>
This is a non-standard part. Use the 110mm version if you prefer a more compact mounting plate.
</Note>
