## Function

The 110mm Tie Down and Suction Plate provides versatile toy mounting options. This component:

- Offers a 110mm diameter mounting surface
- Features slots for tie-down straps to secure toys
- Provides a smooth surface for suction cup attachment
- Supports dual mounting methods for different toy types

<Note>
This is a non-standard part. Use the 135mm version if you need a larger mounting surface.
</Note>
