## RADR (Research and Desire Remote)

The RADR is the official wireless controller for the OSSM. With Bluetooth connectivity, you can control your OSSM without being tethered by cables.

The RADR is fully open source. If you have access to the required components and tools, you can build one yourself using the designs in this repository.
</Tip>

### Getting a RADR

You have two options for obtaining a RADR remote:

1. **Build it yourself** — Access the open-source designs and firmware in the Hardware section of this documentation
2. **Purchase pre-built** — Check the Kinky Makers Discord for availability from official builders
