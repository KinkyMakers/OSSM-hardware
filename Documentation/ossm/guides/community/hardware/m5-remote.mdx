title: "M5 Remote"
description: "A wireless touchscreen remote for OSSM with four rotary encoders"
---

The M5 Remote is a community-developed wireless controller built on the M5Stack platform. It features four rotary encoders for direct control of speed, depth, stroke, and sensation, plus a touchscreen display for visual feedback.
